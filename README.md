## Setup/Installation Requirements

#### You will need to clone the repository:

* Open up terminal
* Copy and paste this:
* cd ~desktop
* git clone https://github.com/Griflongtin/Tamagotchi-React.git
* cd Tamagotchi-React
* npm i
* npm run start
